# 新增怪兽图片说明

## 需要添加的图片文件

请将以下图片文件放置到 `public/assets/images/` 文件夹中：

### 1. virus.png (病毒兽)
- **建议尺寸**: 400x400px 或更高
- **格式**: PNG（支持透明背景）
- **设计建议**: 
  - 紫色/绿色的病毒形态
  - 带有尖刺或突变特征
  - 充满侵略性的视觉效果

### 2. bacteria.png (细菌兽)
- **建议尺寸**: 400x400px 或更高
- **格式**: PNG（支持透明背景）
- **设计建议**:
  - 圆形或椭圆形的细菌形态
  - 紫色/蓝色调
  - 分裂或增殖的视觉效果
